      # Usage

      $ npx create-rubik-app my-app

      # Command

        rubik build          Build.
  rubik check-package  Check the package.json file is changed from pre version.
  rubik commitlint     Commit lint.
  rubik lint           Eslint.
  rubik serve          Start a http server.
  rubik stylelint      Style lint.
